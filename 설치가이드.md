# KH Project 설치 가이드

## 사전 요구사항

### 1. Python 설치 (이미 설치된 경우 건너뜀)
1. https://www.python.org/downloads/ 접속
2. "Download Python 3.x.x" 버튼 클릭하여 설치
3. **설치 시 "Add Python to PATH" 체크 필수**
4. 설치 완료 후 PowerShell에서 확인:
   ```
   python --version
   ```

### 2. Node.js 설치 (이미 설치된 경우 건너뜀)
1. https://nodejs.org/ 접속
2. "LTS" 버전 다운로드 및 설치
3. 설치 완료 후 PowerShell에서 확인:
   ```
   node --version
   ```

## 설치

## Claude Desktop MCP 등록

### 1. 법제처 API 키 발급 (국가법령 서버용)
1. https://open.law.go.kr 접속 → 회원가입 → API 신청
2. 발급받은 키를 메모해둠

### 2. Claude Desktop 설정 파일 열기

아래 경로의 파일을 메모장으로 열기:
```
C:\Users\[사용자이름]\AppData\Roaming\Claude\claude_desktop_config.json
```

> AppData 폴더가 안 보이면: 탐색기 주소창에 `%APPDATA%\Claude` 직접 입력

### 3. 설정 추가

파일 내용을 아래와 같이 수정 (`[압축해제경로]`와 `[API키]` 부분을 실제 값으로 교체):

```json
{
  "mcpServers": {
    "ADD_규정DB": {
      "command": "python",
      "args": ["[압축해제경로]\\kh_rules\\mcp_server.py"],
      "cwd": "[압축해제경로]\\kh_rules"
    },
    "국가법령": {
      "command": "node",
      "args": ["[압축해제경로]\\korean-law-mcp-kh\\build\\index.js"],
      "env": {
        "LAW_OC": "[API키]",
        "NODE_OPTIONS": "--use-system-ca"
      }
    }
  }
}
```

**예시** (압축 해제 경로가 `C:\tools\khproject_dist` 인 경우):
```json
{
  "mcpServers": {
    "ADD_규정DB": {
      "command": "python",
      "args": ["C:\\tools\\khproject_dist\\kh_rules\\mcp_server.py"],
      "cwd": "C:\\tools\\khproject_dist\\kh_rules"
    },
    "국가법령": {
      "command": "node",
      "args": ["C:\\tools\\khproject_dist\\korean-law-mcp-kh\\build\\index.js"],
      "env": {
        "LAW_OC": "abc123여기에API키입력",
        "NODE_OPTIONS": "--use-system-ca"
      }
    }
  }
}
```

> 이미 다른 MCP 서버가 등록되어 있다면 `"mcpServers"` 안에 위 내용을 추가로 넣으면 됩니다.

### 4. Claude Desktop 재시작

설정 저장 후 Claude Desktop을 완전히 종료했다가 다시 실행.

### 5. 확인

Claude Desktop 채팅창 좌측 하단에 MCP 도구 아이콘이 표시되면 정상 등록 완료.
